# Video 8: JNexus on iOS and macOS

**Length:** 5:00  
**Audience:** Apple ecosystem users  
**Goal:** Tour both iOS and macOS apps, highlighting platform-specific features

---

## Scene 1: Introduction (0:00 - 0:15)

**Visual:** Intro card, then iPhone and Mac side by side.

**Voiceover:**
> "JNexus has native apps for iOS, iPadOS, and macOS -- all built with SwiftUI and sharing 95 percent of their code. Let's tour both platforms and see what makes each one special."

---

## Scene 2: Installation (0:15 - 0:45)

**Visual:** GitHub releases page showing IPA and DMG downloads.

**Voiceover:**
> "Download the iOS app as an IPA file or the macOS app as a DMG from the GitHub releases page. iOS requires version 16.0 or higher; macOS requires Ventura 13.0 or higher."

**Visual:** Show installing on each platform.

> "On iOS, install via TestFlight or AltStore. On macOS, open the DMG and drag JNexus to your Applications folder."

---

## Scene 3: iOS Interface Tour (0:45 - 2:15)

**Visual:** iPhone showing the JNexus app.

**Voiceover:**
> "On iPhone, JNexus uses a tab bar at the bottom for navigation: List, Search, Stats, and Settings."

**Visual:** Show the List tab with components.

> "The List tab shows your repository components. Pull down to refresh, or tap the Refresh button. Tap a component to see its full metadata. Swipe left to reveal the delete action."

**Visual:** Show the Search tab with filters.

> "The Search tab provides the same advanced filters as other platforms: size range, date range, file extension, name pattern, and path regex."

**Visual:** Show the Stats tab.

> "The Stats tab displays comprehensive repository analytics, optimized for the phone screen."

**Visual:** Show the Settings tab.

> "And Settings is where you configure your Nexus credentials. They're stored securely in the iOS Keychain with hardware-backed AES-256 encryption."

---

## Scene 4: iPad Split View (2:15 - 2:45)

**Visual:** iPad in landscape showing the split view layout.

**Voiceover:**
> "On iPad, JNexus takes advantage of the larger screen with a sidebar navigation pattern. The sidebar shows your repositories and navigation items. The detail area shows the selected view."

**Visual:** Show rotating to landscape mode.

> "In landscape, the split view is always visible. In portrait, swipe from the left edge to reveal the sidebar. iPad also supports Split View multitasking -- run JNexus alongside your browser or notes app."

---

## Scene 5: macOS Sidebar Navigation (2:45 - 3:45)

**Visual:** macOS app showing the sidebar and main content area.

**Voiceover:**
> "The macOS app uses a sidebar navigation pattern that's native to the Mac experience. Repositories appear in the sidebar, and the main area shows the selected view."

**Visual:** Show the menu bar integration.

> "Full menu bar integration gives you access to all commands. The File menu handles profiles, Edit provides standard clipboard operations, and the View menu controls the sidebar and toolbar."

**Visual:** Show multiple windows.

> "macOS supports multi-window. Open separate windows for different repositories or profiles, and work with them side by side."

---

## Scene 6: Keyboard Shortcuts (3:45 - 4:15)

**Visual:** Show keyboard shortcuts being used on macOS.

**Voiceover:**
> "On macOS, keyboard shortcuts let you work efficiently. Command-L opens the List view, Command-R refreshes, Command-F focuses the filter field. These shortcuts are standard Mac conventions."

**Visual:** Show the keyboard shortcut reference (in the menu bar or a help overlay).

> "All shortcuts are visible in the menu bar. Learn the ones you use most and you'll rarely need the mouse."

---

## Scene 7: Keychain Integration (4:15 - 4:45)

**Visual:** macOS Keychain Access showing the JNexus entry, or the Settings view on iOS.

**Voiceover:**
> "On both iOS and macOS, credentials are stored in the system Keychain. On iOS, this means hardware-backed encryption on devices with a Secure Enclave. On macOS, credentials integrate with the Keychain Access app and support biometric unlock with Touch ID."

> "If you use iCloud Keychain, your credentials can sync across your Apple devices -- but this is optional and can be disabled."

---

## Scene 8: Wrap-Up (4:45 - 5:00)

**Visual:** Side-by-side of iPhone and Mac showing the same repository.

**Voiceover:**
> "Whether you're at your desk with a Mac or on the go with your iPhone, JNexus provides a consistent, native experience for managing your Nexus repositories. Check the links in the description for download and setup instructions."

**Visual:** Fade to outro card.

---
