## Contributing Screenshots

To capture screenshots:

1. **Swing/AWT**: Use your OS screenshot tool or Java's Robot class
2. **Terminal**: Use `script` command or terminal screenshot feature
3. **Android**: Use Android Studio's screen capture or `adb shell screencap`
4. **iOS**: Use Xcode Simulator's screenshot feature (Cmd+S)
5. **macOS**: Use Cmd+Shift+4 for selection or Cmd+Shift+3 for full screen

Guidelines:
- Use PNG format for all screenshots
- Resize to a reasonable width (800-1200px)
- Use sample/demo data (never real credentials or production data)
- Crop to show only the relevant window/screen
