JNexus is a tool for managing components in Sonatype Nexus Repository Manager.

Features:
* List and search components with regex filtering
* Advanced filters: size range, date range, file extension, name pattern
* Delete components with dry-run and confirmation safety
* Repository statistics: size distribution, file types, age analysis
* Encrypted credential storage (AES-256-GCM)
* Material Design 3 UI
* Profile-based configuration for multiple environments

JNexus is free and open source software licensed under the Apache License 2.0.
